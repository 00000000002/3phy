phy.set()

set physics engine option

Code Example

phy.set({ fps:60, substep:2, gravity:[0,-9.81,0] })

Command

phy.set( [param:Object parameters] )

Parameters

[param:Int fps] — choose engine number of frame by seconde. Default is 60
[param:Int substep] — how many pass loop will make, is increase accuraty but also calculation. Default is 2
[param:Array gravity] — the gravity vector [x,y,z]
