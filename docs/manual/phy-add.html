phy.add()

add object to simulation

Code Example

phy.add({ name:'yo', type:'box', density:1, size:[2,2,2], pos:[0,2,0], rot:[0,0,0] })

Command

phy.add( [param:Object parameters] )

phy.add( [param:Array list] )

Parameters

[param:String name] — name of object is automatic if not set
[param:String type] — type of object
[param:Float density] — is the density of object mass = density * volume
[param:Array size] — size of object
[param:Array pos] — position of object
[param:Array rot] — rotation of object in degrée
