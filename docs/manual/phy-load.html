phy.load()

load resource

Code Example

phy.load( 'dices.glb', onComplete ) phy.load( ['dices.glb', 'yo.jpg'], onComplete )

Command

phy.load( [param:String name], [param:Function callback] )

phy.load( [param:Array list], [param:Function callback] )

Parameters

[param:String name] — load one object by name
[param:Array list] — load list of objects name
[param:Function callback] — on loading complete
