phy.remove()

remove object from simulation

Code Example

phy.remove('object') phy.remove(['object1', 'object2', 'object3'])

Command

phy.remove( [param:String name] )

phy.remove( [param:Array list] )

Parameters

[param:String name] — remove one object by name
[param:Array list] — remove list of objects name

Note

phy auto remove object if you add new object with same name
